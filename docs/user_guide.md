# Guide Utilisateur - Borne d'Arcade IUT

Bienvenue sur la borne d'arcade du département informatique !

## 🎮 Contrôles de la Borne

La borne est équipée de joysticks et de boutons d'arcade classiques. Voici la correspondance matérielle détaillée :

### Joueur 1 (Gauche)
* **Joystick** : Flèches directionnelles (Haut, Bas, Gauche, Droite)
* **Boutons d'action (Bas)** : **A** (`f`), **B** (`g`), **C** (`h`)
* **Boutons d'action (Haut)** : **X** (`r`), **Y** (`t`), **Z** (`y`)

### Joueur 2 (Droite)
* **Joystick** : Haut (`o`), Bas (`l`), Gauche (`k`), Droite (`m`)
* **Boutons d'action (Bas)** : **A** (`q`), **B** (`s`), **C** (`d`)
* **Boutons d'action (Haut)** : **X** (`a`), **Y** (`z`), **Z** (`e`)

## 🕹️ Navigation dans le Menu Principal

1. **Choisir un jeu** : Utilisez le **Joystick du Joueur 1 (Haut / Bas)** pour faire défiler les jeux disponibles dans le carrousel.
2. **Lancer un jeu** : Appuyez sur le **Bouton A du Joueur 1** (`f`).
3. **Quitter la borne** : Appuyez sur le **Bouton Z du Joueur 1** (`y`). Une boîte de dialogue *"Voulez-vous vraiment quitter ?"* apparaîtra. Sélectionnez "OUI" ou "NON" avec le joystick (Gauche/Droite) et validez avec le **Bouton A**.

## 🏆 High Scores
À la fin d'une partie, si vous battez un record, un écran de saisie apparaîtra :
* Utilisez le **Joystick Haut/Bas** pour faire défiler les lettres de l'alphabet.
* Utilisez le **Joystick Gauche/Droite** pour changer de case (3 lettres maximum).
* Appuyez sur le **Bouton A** sur le dernier caractère pour valider et enregistrer votre score.
